


Tanto en historial, como en auditoria añade una columna después de fecha con el número de respaldo.
La idea es identificar rápidamente el número de respaldo de una operación.

Si para aplicar cambios el workspace está con algúb lock que no te permite realizar cambios, puedes identificar
el PID y detener el proceso que está bloqueando el workspace.
De esta manera, podrás realizar los cambios necesarios sin problemas.
